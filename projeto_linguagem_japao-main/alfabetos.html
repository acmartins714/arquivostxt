Alfabetos

O Hiragana é usado para palavras de origem japonesa, partículas e elementos gramaticais, sendo ideal para iniciantes. Exemplo: わたしはがくせいです (Watashi wa gakusei desu - "Eu sou estudante").


O Katakana é utilizado para palavras de origem estrangeira, nomes internacionais e onomatopeias. Exemplo: パン (pan - "pão").


Já o Kanji é composto por caracteres de origem chinesa e é essencial no japonês, combinado com o hiragana e katakana para formar palavras e expressar ideias complexas. Ele é indispensável para leitura avançada e compreensão plena do idioma. Exemplo: 日本語を勉強します (Nihongo o benkyou shimasu - "Eu estudo japonês").
